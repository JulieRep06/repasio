# Repasio — Planificateur de repas familial

Application web de planification de repas avec :
- Planning hebdomadaire intelligent
- Gestion du stock et des péremptions
- Liste de courses automatique
- Budget maîtrisé
- Génération automatique de menus (saisonnalité, stock, budget)

## Stack
- Frontend : HTML/CSS/JS (single-page)
- Backend : Supabase (PostgreSQL + Auth + RPC)
- Hébergement : GitHub Pages

## Déploiement
Le site se déploie automatiquement via GitHub Pages sur la branche `main`.
